# Software Verification and Validation - Assignment 1

## Project Description
This project focuses on the verification and validation of a "Create New Account" feature. 
The goal is to implement unit tests using **Equivalence Partitioning** and **Boundary Value Analysis** techniques.

## Features to Test
* First Name / Last Name
* E-mail
* Date of Birth
* Password / Confirm Password

## Tech Stack
* **Language:** Python
* **Testing Framework:** Pytest
* **CI/CD:** GitHub Actions
